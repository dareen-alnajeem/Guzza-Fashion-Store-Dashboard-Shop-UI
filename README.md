Guzza – Fashion Store Dashboard & Shop UI

This project is a modern and responsive fashion e-commerce website that includes a sleek landing page, product filtering system, and interactive UI components such as scrollable sections and a dynamic dashboard.

The website is designed to provide a smooth and user-friendly shopping experience with a focus on modern UI/UX principles.

⸻

 Features
 •  Interactive hero section with product showcase slider
 •  Category filtering system (products change based on user selection)
 •  Fashion product grid with smooth scrolling experience
 •  Simple dashboard-style layout for managing or viewing items
 •  Clean and modern UI design
 •  Fully responsive across all devices

⸻

 Functionalities
 • Users can browse products through horizontal scrolling sections
 • Products can be filtered dynamically using the navigation menu
 • Smooth transitions between categories
 • Structured layout similar to real-world e-commerce platforms

⸻

 Purpose

This project was built to practice front-end development skills including layout design, interactive filtering systems, and responsive UI development inspired by modern online fashion stores.

 Tech Used
